# Simpix

**Simpix** is a web application for searching and browsing images, videos, and collections using the **Pexels API**.

## Features

- Search for images, videos, and collections via the Pexels API
- Media filtering and preview
- Download images and videos with selectable sizes/resolutions
- Favorites system stored in **Local Storage**

## Built With

- HTML5
- CSS
- JavaScript

## Installation / Setup

1. Clone the repository
2. Configure the Pexels API key:
   - Open js/api_configure.js
   - Replace {YOUR-PEXELS-API-KEY} with your Pexels API key
3. Open index.html in your browser
